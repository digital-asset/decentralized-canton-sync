.. _type-splice-svonboarding-svonboardingrequest-71856:

**template** `SvOnboardingRequest <type-splice-svonboarding-svonboardingrequest-71856_>`_

  Template used by the SVs to collect confirmations for the onboarding of an SV candidate\.
  The existence of this contract triggers SV automation that creates confirmation contracts for
  the candidate if the token is a valid ``SvOnboardingToken`` and matches an ``ApprovedSvIdentity``\.

  Signatory\: dso

  .. list-table::
     :widths: 15 10 30
     :header-rows: 1

     * - Field
       - Type
       - Description
     * - candidateName
       - `Text <https://docs.daml.com/daml/stdlib/Prelude.html#type-ghc-types-text-51952>`_
       - Human\-readable name of the SV candidate\. Must match the one in ``ApprovedSvIdentity``!
     * - candidateParty
       - `Party <https://docs.daml.com/daml/stdlib/Prelude.html#type-da-internal-lf-party-57932>`_
       - PartyId of the candidate SV party\.
     * - candidateParticipantId
       - `Text <https://docs.daml.com/daml/stdlib/Prelude.html#type-ghc-types-text-51952>`_
       - ParticipantId of the candidate SV\.
     * - token
       - `Text <https://docs.daml.com/daml/stdlib/Prelude.html#type-ghc-types-text-51952>`_
       - An encoded and signed ``SvOnboardingToken`` that confirms the candidate's identity\.
     * - sponsor
       - `Party <https://docs.daml.com/daml/stdlib/Prelude.html#type-da-internal-lf-party-57932>`_
       - The established SV node that created this contract\.
     * - dso
       - `Party <https://docs.daml.com/daml/stdlib/Prelude.html#type-da-internal-lf-party-57932>`_
       -
     * - expiresAt
       - `Time <https://docs.daml.com/daml/stdlib/Prelude.html#type-da-internal-lf-time-63886>`_
       - When this contract can be archived even if the onboarding did not succeed\.

  + **Choice** Archive

    Controller\: dso

    Returns\: ()

    (no fields)
