.. _type-splice-round-openmininground-90060:

**template** `OpenMiningRound <type-splice-round-openmininground-90060_>`_

  A mining round for which new rewards can be registered\.
  Note that multiple mining rounds can be open at the same time to give some
  time for propagating a new open round, while still registering rewards
  against the previous open round\.

  Signatory\: dso

  .. list-table::
     :widths: 15 10 30
     :header-rows: 1

     * - Field
       - Type
       - Description
     * - dso
       - `Party <https://docs.daml.com/daml/stdlib/Prelude.html#type-da-internal-lf-party-57932>`_
       -
     * - round
       - :ref:`Round <type-splice-types-round-16181>`
       -
     * - amuletPrice
       - `Decimal <https://docs.daml.com/daml/stdlib/Prelude.html#type-ghc-types-decimal-18135>`_
       -
     * - opensAt
       - `Time <https://docs.daml.com/daml/stdlib/Prelude.html#type-da-internal-lf-time-63886>`_
       - Time after which transfers can use this mining round
     * - targetClosesAt
       - `Time <https://docs.daml.com/daml/stdlib/Prelude.html#type-da-internal-lf-time-63886>`_
       - Time when this round is expected to be closed as part of standard DSO\. The round SHOULD NOT be archived before this time\.
     * - issuingFor
       - `RelTime <https://docs.daml.com/daml/stdlib/DA-Time.html#type-da-time-types-reltime-23082>`_
       - Timepoint on the issuance curve that was used to determing the issuance configuration for this round\.
     * - transferConfigUsd
       - :ref:`TransferConfig <type-splice-amuletconfig-transferconfig-45691>` :ref:`USD <type-splice-amuletconfig-usd-42527>`
       - Configuration determining the fees and limits in USD for Amulet transfers
     * - issuanceConfig
       - :ref:`IssuanceConfig <type-splice-issuance-issuanceconfig-93012>`
       - Configuration for issuance of this round\.
     * - tickDuration
       - `RelTime <https://docs.daml.com/daml/stdlib/DA-Time.html#type-da-time-types-reltime-23082>`_
       - Duration of a tick, which is the duration of half a round\.

  + **Choice** Archive

    Controller\: dso
